A plugin to categorize users with custom taxonomies.

== Description ==
This plugin allows you to categorize users using custom taxonomies. It includes an admin interface for managing user tags, filtering users by tags, and a dynamic AJAX-powered search for user tags.

== Installation ==
1. Upload the `user-tags` folder to the `wp-content/plugins` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
